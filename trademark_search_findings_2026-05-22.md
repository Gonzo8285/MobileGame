## What this search CAN'T tell us

Discovery search via web is limited. Specifically:

- **USPTO trademark register** — needs an interactive query at `tmsearch.uspto.gov`. The web index doesn't surface live trademark records reliably. Paul (or a solicitor) should run the actual search before any UK / US / EU trademark application.
- **UKIPO trademark register** — same. Run at `www.gov.uk/search-for-trademark`.
- **EUIPO trademark register** — same. Run at `euipo.europa.eu/eSearch/`.
- **WIPO Global Brand Database** — same. Run at `branddb.wipo.int`.
- **Older paper / pre-internet IP** (a novel from 1992, a tabletop module from 2005, etc) — discovery search misses anything pre-Google-indexed.
- **In-development indie names** — anything not yet published / announced is invisible.

## Recommended next steps for Paul

### Before any external print of "Gallowfell"

1. **Run UKIPO trademark search** at `https://www.gov.uk/search-for-trademark` — 5 min. Search for "Gallowfell" in classes 9 (software / mobile apps / downloadable games) and 41 (entertainment services).
2. **Run USPTO trademark search** at `https://tmsearch.uspto.gov` — 10 min. Same classes (US class 9 + 41 equivalent — IC 009 + IC 041).
3. **Run EUIPO eSearch** at `https://euipo.europa.eu/eSearch/` — 5 min. Same classes.
4. **Run WIPO Global Brand DB** at `https://branddb.wipo.int` for international cross-reference — 5 min.

Total: ~25 min of clean-room searching. If any hits surface in those four, surface them in chat and we triage.

### Before soft launch (regardless of trademark)

5. **Reserve store names** on Steam, App Store, Play Store as soon as the soft-launch date is set. Even without a registered trademark, store-name reservation gives you de facto priority on the platform that matters most for distribution.
6. **Buy the domains** — `gallowfell.com`, `thecurseofgallowfell.com`, `gallowfellgame.com` if available. Cheap insurance; check via Namecheap / 1-2-3-Reg.
7. **Snag social handles** on Twitter/X, Bluesky, Instagram, TikTok, YouTube — also cheap insurance.

### When to file an actual trademark application

- After soft launch, before broader release. Trademark applications cost £200-300 per class in the UK, more in US/EU.
- Best filed by an IP solicitor (~£500-800 for a single-class UK filing including their fee). Not urgent until commercial volume justifies it.
